FORM ETİKETLERİ

Form etiketleri form etiketi içinde kullanılırlar.

"input" etiketi ile bilgi girilmesi gereken kutucuklar oluşturulur.

    Aşağıdaki input seçenekleri mevcuttur.

  • text
  • password
  • file
  • checkbox

    label etiketi ve input etiketi


  • "label" etiketi ile kutucukların yanına ilgili bilgi girilmesi sağlanır.

  • input etiketini label etiketinin içine alarak yazıya tıklandığında da kutucuğa fokuslanması sağlanabilir.

  • bunun bir diğer yöntemi de önce label etiketi oluşturularak for kısmına yazılan veri,
    daha sonra aşağı input etketi içinde oluşturulan "id" etiketiile aynı olmasıdır.

  • özetle label içindeki "for" ve input içindeki "id" aynı olmalıdır.
  • radio çoklu seçeneklerden sadece birinin seçmesi sağlanabilir.
    Bunu sağlamak için input etiketleri içindeki "name" seçeneklerde aynı olmalıdır.
    Hangi değerin seçili olduğumum belli olması için kendi kutusunun "value" değeri olarak girilmelidir.

  • gene bu input etiketlerini label etiketleri içine alarak metne
    tıklandığında kutucuklara fokuslanması sağlanabilir.
  • "Checkbox" etiketi ile birden çok kutucuğu işaretlemek mümkündür.

  • checked input içine yazılarak ön seçili olması sağlanabilir.

  • required ile boşlukların mecbur seçilmesi sağlanabilir.

  • date/week/month ile tarih seçilebilir.
  • input type tel ile telefon girilmesi sağlanabilir.

  • input type url ile https ile internet adresi girilmesi zorunlu tutulur.

  • placeholder ile kutucuk içinde silik veri örnek bilgi gereklilik yazılabilir.

  • dosya seçerken bazı dosyaların seçilmesini sağlamak bazılarını engelleyebilmek için
    input type="file" içinde accept="image/*"(burada dosya şekli resim uzantısı farketmez demek)
    şeklinde yazılır.
  • autocoplete onn ve off ile kutucuğun otomatik dolması veya kapanması sağlanır.

      Select

    • select etiketi ile kutucuğa tıklandığında çoklu veriden birinin seçilmsesini amaçlanır.
      select içine multiple yazılarak seçeneklerin çoklu olması da mümkündür.

    • select içindeki verilri "optgroup" ile gruplayabilirz.
